# AI Reminder Optimization Environment

## Overview

This environment simulates a real-world reminder system where an AI agent must decide when to notify users.

## Features

- User Profiles: Executive, Procrastinator
- Context Awareness: busy level, importance, deadline
- Reward System: based on smart decision making

## Actions

- 0: Wait
- 1: Send Reminder

## Goal

Maximize user task completion while minimizing annoyance.

## Score

Rewards range from 0.0 to 1.0
